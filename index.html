Helmet And Number Plate Detection

In this project, the Helmet and number plate of the bikers are detected, and the message is displayed on the screen according to the condition. if the biker is wearing a helmet it shows Helmet on the screen else it shows No Helmet on the screen.

Tech Stack:- Python, Machine learning

Sensor data into Cloud Server for Analytics

In this project, the temperature and humidity of the environment are sensed through the DHT11 sensor at a frequent intervals of time and the data is uploaded to the ThingSpeak server (Cloud) via API. where the data is represented in graphical form.

Tech Stack:- Embedded C language, Arduino.

Quake Report (Android Application)

This is an app that shows updates on the latest earthquakes around the world. where the user has the option to choose the earthquakes to be visible based on the magnitude of earthquakes. this project uses API from USGS(United States Geological Survey) website.

Tech Stack:- Java, Data structures, UI design, Android Studio.
